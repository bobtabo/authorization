# TypeScript + Hono

認可サーバー API の **TypeScript / Hono** 実装用ディレクトリです。

- **言語:** TypeScript
- **ランタイム:** Node.js または [Bun](https://bun.sh/) / [Deno](https://deno.com/)（実装時に決定）
- **フレームワーク:** [Hono](https://hono.dev/)

API 仕様はリポジトリルートの [`docs/api-spec`](../docs/api-spec/README.md) を参照してください。
